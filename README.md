HelloWorld is a simple "hello world" style class for testing a simple Composer-based class.

## Requirements

* PHP 5.3 or higher

## Installation

Add the following to your `composer.json`:

```json
"require": {
	"CodeDruids/hello-world": "1.*"
},
"repositories": [
	{
		"type": "vcs",
		"url": "git://github.com/CodeDruids/hello-world.git"
	}
],
```
